Memory Game

Available

Description

Train your memory and beat the competition in our exciting Memory Game challenge. Participants will test their short-term memory capacity, recall speed, and pattern recognition abilities through a series of increasingly difficult memory tasks. Can you remember more than your competitors?

Registration: Open
Required IQ Test Grade: 100+
Start Date: June 10 2025
Location: Online
Gender: All
Game Type: Individual
Time Limit: 45 minutes
Difficulty: Beginner to Advanced

Awards

  • 1st Place: 50000DA + Gold Medal
  • 2nd Place: 25000DA + Silver Medal
  • 3rd Place: 10000DA + Bronze Medal
  • Certificates for all participants

Rules

  • Participants must complete all memory challenges within the time limit
  • No external aids or memory techniques using physical tools are allowed
  • Each level must be completed before proceeding to the next
  • Points are awarded based on accuracy, speed, and difficulty level completed
  • Judges' decisions are final
